# 🚀 Project 1: Emotion Prediction from Text

This project focuses on **predicting emotions from text using NLP techniques**.

### 📌 Objective

To build a machine learning model that can analyze text input and classify the **emotion expressed in the sentence**.

### 🧠 Emotions Predicted

Examples may include:

* Joy 😊
* Sadness 😢
* Anger 😡
* Fear 😨
* Love ❤️
* Surprise 😲

### 🔧 Techniques Used

* Text Cleaning
* Tokenization
* Stopword Removal
* Bag of Words
* Feature Extraction
* Machine Learning Model Training

### 🛠 Libraries Used

* Python
* NumPy
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn
* NLTK

# 📊 Bag of Words Implementation

The repository also includes a notebook that demonstrates the **Bag of Words (BoW) technique**, one of the most fundamental methods in NLP.

### What it covers:

* Converting text into numerical vectors
* Vocabulary creation
* Word frequency representation
* Feature matrix generation

This helps in understanding how **machine learning models process textual data**.
